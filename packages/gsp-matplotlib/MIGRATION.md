# gsp-matplotlib migration provenance

Source baseline: `vispy/GSP_API@463d34d1d6560f045e5c40af594372d0fea93ab5`.

Formal renderer, query, guide, layout, navigation, tiled-image, transform, capability, and color
mapping files are curated from source tree `2a128e69f5fc83d1e6406dbc1688454b8ca55a78` using the
selection in `migration-manifest.json`. The initializer, plugin, and session are derived new files.
